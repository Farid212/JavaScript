# Add Border

Given a rectangular matrix of chararcters, add a border of asterisks(*) to it.

### Example

for 

```
picture = ["abc", "ded"]
```

the output should be

```
addBorder(picture) = ["*****",
                    "*abc*",
                    "*ded*",
                    "*****"]
```

### Hints

- concat()
- unshift()
- push()

### Input/Output

- **[time limit] 4000ms (js)**
- **[input] array.string picture**

a non-empty array of non-empty equal-length strings.
